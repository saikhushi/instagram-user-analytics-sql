\# **Instagram User Analytics SQL Project**



\## ***Overview***

This project analyzes user engagement on a social media platform similar to Instagram.



\## **Database Features**

\- Users

\- Posts

\- Likes

\- Comments

\- Followers



\## **Analysis Performed**

\- Most active users

\- Most liked posts

\- Top influencers

\- Users who never posted

\- Posts with no likes



\## **Tools Used**

SQL Server Management Studio (SSMS)
